Redis вҖӢ

Redis з®Җд»Ӣ вҖӢ

Redis жҳҜ Remote Dictionary ServerпјҲиҝңзЁӢеӯ—е…ёжңҚеҠЎеҷЁпјүзҡ„зј©еҶҷпјҢжҳҜдёҖдёӘејҖжәҗзҡ„дҪҝз”Ё ANSI C иҜӯиЁҖзј–еҶҷгҖҒж”ҜжҢҒзҪ‘з»ңгҖҒеҸҜеҹәдәҺеҶ…еӯҳдәҰеҸҜжҢҒд№…еҢ–зҡ„ж—Ҙеҝ—еһӢгҖҒKey-Value ж•°жҚ®еә“пјҢе№¶жҸҗдҫӣеӨҡз§ҚиҜӯиЁҖзҡ„ APIгҖӮ

Redis е®үиЈ… вҖӢ

еңЁ Ubuntu дёҠе®үиЈ… Redis еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

bash
sudo apt update
sudo apt install redis-server

Redis й…ҚзҪ® вҖӢ

Redis еҹәжң¬е‘Ҫд»Ө вҖӢ

Redis дёӯзҡ„еҸҳйҮҸе‘Ҫд»Өдё»иҰҒжңүд»ҘдёӢеҮ дёӘпјҡ

  • SETпјҡи®ҫзҪ®жҢҮе®ҡ key зҡ„еҖј
  • GETпјҡиҺ·еҸ–жҢҮе®ҡ key зҡ„еҖј
  • DELпјҡеҲ йҷӨжҢҮе®ҡ key
  • FLUSHALLпјҡеҲ йҷӨжүҖжңү key
  • EXISTSпјҡеҲӨж–ӯ key жҳҜеҗҰеӯҳеңЁ
  • EXPIREпјҡи®ҫзҪ® key зҡ„иҝҮжңҹж—¶й—ҙ
  • TTLпјҡиҺ·еҸ– key зҡ„иҝҮжңҹж—¶й—ҙ
  • PERSISTпјҡз§»йҷӨ key зҡ„иҝҮжңҹж—¶й—ҙ
  • KEYSпјҡиҺ·еҸ–жүҖжңү keyпјҢеҸҜд»ҘдҪҝз”ЁйҖҡй…Қз¬Ұ
  • RENAMEпјҡйҮҚе‘ҪеҗҚ key
  • TYPEпјҡиҺ·еҸ– key зҡ„зұ»еһӢ
  • APPENDпјҡиҝҪеҠ еӯ—з¬ҰдёІ
  • STRLENпјҡиҺ·еҸ–еӯ—з¬ҰдёІй•ҝеәҰ
  • INCRпјҡиҮӘеўһ

еҹәжң¬ж“ҚдҪң вҖӢ

Redis дёӯдҪҝз”Ё key-value еӯҳеӮЁж•°жҚ®пјҢSET е‘Ҫд»Өз”ЁдәҺи®ҫзҪ®жҢҮе®ҡ key зҡ„еҖјгҖӮ

bash
SET key value
# е®Ңж•ҙиҜӯжі•
SET key value [NX|XX] [GET] [EX seconds|PX milliseconds|EXAT unix-time-seconds|PXAT unix-time-milliseconds|KEEPTTL]

дҪҝз”Ё GET е‘Ҫд»ӨиҺ·еҸ– key зҡ„еҖјпјҡ

bash
GET key

Redis зҡ„ key жҳҜеҢәеҲҶеӨ§е°ҸеҶҷзҡ„пјҢжүҖд»Ҙ key е’Ң Key жҳҜдёӨдёӘдёҚеҗҢзҡ„ keyгҖӮ

Redis й»ҳи®ӨдҪҝз”Ёеӯ—з¬ҰдёІеӯҳеӮЁй”®е’ҢеҖјпјҢиҖҢдё”жҳҜдәҢиҝӣеҲ¶е®үе…Ёзҡ„пјҢжүҖд»Ҙ value еҸҜд»ҘжҳҜеӯ—з¬ҰдёІгҖҒж•°еӯ—гҖҒеёғе°”еҖјгҖҒеәҸеҲ—еҢ–еҜ№иұЎзӯүгҖӮ

иҝҮжңҹж—¶й—ҙ вҖӢ

Redis дёӯеҸҜд»Ҙдёә key и®ҫзҪ®иҝҮжңҹж—¶й—ҙпјҢдҪҝз”Ё EXPIRE е‘Ҫд»Өи®ҫзҪ® key зҡ„иҝҮжңҹж—¶й—ҙпјҡ

bash
EXPIRE key seconds

дҪҝз”Ё TTL е‘Ҫд»ӨиҺ·еҸ– key зҡ„иҝҮжңҹж—¶й—ҙпјҡ

bash
TTL key

дҪҝз”Ё PERSIST е‘Ҫд»Өз§»йҷӨ key зҡ„иҝҮжңҹж—¶й—ҙпјҡ

bash
PERSIST key

д№ҹеҸҜд»ҘеңЁи®ҫзҪ®зҡ„ж—¶еҖҷзӣҙжҺҘи®ҫзҪ®иҝҮжңҹж—¶й—ҙпјҡ

bash
SET key value EX seconds
SETEX key seconds value
