macOS 的配置 ​

通讯录和信息图标 ​

Github 上的开源项目 metowolf/vCards 提供了中国黄页的配置方法，通过订阅 CardDAV 服务自动导入并支持自动更新。

需要设备支持 CardDAV 配置：

  • iOS/iPadOS：「设置」-「通讯录」-「账户」-「添加账户」-「其他」-「添加 CardDAV 账户」

  • macOS：「通讯录」-「设置」-「账户」-「其他通讯录账户」

CardDAV 服务器配置如下：

  • 服务器：vcards.metowolf.com
  • 用户名：cn
  • 密码：cn 或任意填写

⚠️ 注意：默认 iOS 获取新资料的方式为「自动」，该情况只会在连接电源和 WLAN 的情况下才会进行数据推送，可以适当等待一段时间。

当然也可以通过自建 CardDAV 服务进行配置，具体可以参考 教程自建

 Finder

 ​

自定义 Finder 工具栏 ​

参考 在 Mac 上自定义“访达”工具栏，可以「文件或 App」添加至 Finder 工具栏，并重新排列工具栏上的项目

  • 添加项目：按住 ⌘/Command 键，然后将项目（「文件或 App」）拖拽“访达”工具栏直到看到绿色加号。
  • 移除项目：按住 ⌘/Command 键，然后将项目拖出工具栏。
  • 重新排列工具栏中的项目：按住 Command 键，然后将项目拖到新位置。

asitop ​

asitop 是一个针对 Apple Silicon 的性能监控命令行工具，基于 macOS 内置工具 powermetrics 开发的

shell
brew install asitop
sudo asitop
